
# Space Invaders

Ce projet est une implémentation simple du jeu **Space Invaders** utilisant la bibliothèque `turtle` en Python. Le but du jeu est de détruire autant d'aliens que possible en contrôlant un canon qui tire des lasers.

## Fonctionnalités

- **Déplacement du Canon** : Utilisez les flèches gauche et droite pour déplacer le canon.
- **Tir de Laser** : Appuyez sur la barre d'espace pour tirer un laser.
- **Aliens** : Les aliens apparaissent aléatoirement en haut de l'écran et descendent vers le bas. Si un alien atteint le bas, la partie est terminée.
- **Score** : Le score est calculé en fonction du nombre d'aliens détruits.
- **Game Over** : Le jeu s'arrête si un alien atteint le niveau du sol.

## Prérequis

- **Python 3.x** : Assurez-vous d'avoir Python 3 installé sur votre machine.
- **Bibliothèque Turtle** : La bibliothèque `turtle` est incluse avec Python et ne nécessite pas d'installation supplémentaire.

## Installation

1. Clonez ce dépôt ou téléchargez les fichiers source.
   ```bash
   git clone https://github.com/votre-utilisateur/space-invaders.git
   cd space-invaders
   ```
2. Exécutez le script Python.
   ```bash
   python space_invaders.py
   ```

## Commandes

- **Flèche Gauche** : Déplacer le canon vers la gauche.
- **Flèche Droite** : Déplacer le canon vers la droite.
- **Barre d'Espace** : Tirer un laser.
- **Touche "q"** : Quitter le jeu.

## Structure du Projet

- `space_invaders.py` : Le fichier principal du jeu.
- `README.md` : Ce fichier, fournissant des informations sur le projet.
